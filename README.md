# 无穷可能的在线桌游模拟器

- 自定义身份牌
- 自定义手牌
- 从手牌中弃牌
- 从弃牌堆中拿牌
- 实时更新的场上情况

# 快速开始

## 克隆仓库
```bash
git clone https://github.com/ProudCarrotG/BoardGameSimulator
```

## 安装依赖
```bash
pip install flask flask-socketio
```

## 运行
```bash
python app.py
```

## 访问
```bash
http://localhost:5000
```
